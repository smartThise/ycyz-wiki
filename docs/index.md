## :material-pencil-plus: 如何提交修改

本站所有页面都来自校友贡献,**你不需要任何编程基础**,只需要一个 GitHub 账号。

### 第一步:点击右上角的"铅笔"图标 ✏️

在任意页面的**右上角**(顶部导航栏),你会看到一个铅笔图标:

```
┌─────────────────────────────┐
│  银川一中 Wiki        GitHub ✏️│   ← 点击这把铅笔!
└─────────────────────────────┘
```

点击它,GitHub 会直接打开**当前这一页**对应的 Markdown 源文件编辑界面。

### 第二步:在线编辑

1. GitHub 会自动进入「编辑文件」界面。
2. 在网页编辑器里直接修改文字(就像改 Word 一样)。
3. 改完之后,在右下角填写一句话说明你改了什么(例如:*修正了三楼食堂的开放时间*)。
4. 点击 **Propose changes**(提交修改建议)。

### 第三步:提交 Pull Request

1. GitHub 会让你「Create pull request」,直接点绿色的 **Create pull request** 按钮。
2. 等待维护者审核通过 ✅ 后,你的修改就会**自动部署上线**!

> :material-alert-circle:{ .lg } **没有账号?**
> 前往 [github.com](https://github.com/signup) 免费注册一个即可参与编辑。
